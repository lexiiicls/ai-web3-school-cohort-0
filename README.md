## 📂 目录结构说明

- `README.md`: 仓库介绍与指引。
- `profile.md`: 个人画像与学习目标。
- `learning-plan.md`: 专属定制的 AI × Web3 学习计划。
- `daily/`: 每日学习日志、笔记与打卡记录。
- `tasks/`: 核心实战任务与练习。
- `experiments/`: AI Agent 编写、提示词测试或链上交互小实验。
- `handbook-feedback/`: 在学习过程中发现的 Handbook 卡点、错别字或改进建议（用于回流共建）。
- `hackathon/`: 未来可能参与的黑客松创意、设计与代码。
- `submissions/`: 提交给课程的终期作业或阶段性里程碑。
- `templates/`:
  - `daily-note.md`: 每日学习打卡笔记模版。
  - `task-note.md`: 任务笔记与复盘模版。
